## 👀 서비스 소개
#### QR 코드 기반 스마트 케이블 관리 웹 앱 플랫폼
##### 케이블 설치 및 제거 과정에서 발생하는 수작업의 번거로움과 오류를 줄이고, 관리 효율성을 극대화하는 스마트 케이블 관리 서비스
<br>

## 📅 프로젝트 기간
2024.08.02 ~ 2024.09.12 (6주)
<br>

## ⭐ 주요 기능
* QR 코드 생성 및 인쇄
* 작업자 선택하여 알림 및 메일 전송
* 실시간 알림
* 엑셀 보고서 생성
* 로그 및 유지보수 내역 필터링
* 채팅형 FAQ
* 관리자 일정 관리
* 차트를 이용한 데이터 시각화
* 자동 로그아웃
<br>
